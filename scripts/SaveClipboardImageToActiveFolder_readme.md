# Save Clipboard Image To Active Folder

Скрипт для AutoHotkey v2, который одной горячей клавишей сохраняет изображение
из буфера обмена в виде PNG-файла. Изображение сохраняется в ту папку, которая
в данный момент открыта в проводнике Windows (Explorer) или в стандартном
диалоговом окне «Открыть файл» любой программы.

## Установка

Нужен установленный autohotkey
[Прямая ссылка для скачивания](https://www.autohotkey.com/download/ahk-v2.exe) (3 Мб)

Скачайте и сохраните файл скрипта [SaveClipboardImageToActiveFolder.ahk](SaveClipboardImageToActiveFolder.ahk)
в любое удобное место на компьютере.

Запустите скрипт двойным щелчком мыши.
При стартре будет предложено добавить скрипт в автозагрузку.
Для удаления из автоагрузки `Ctrl + Win + Del` при запущеном скрипте.

После запуска в области уведомлений (системном трее) появится иконка AutoHotkey,
сигнализирующая о работе скрипта.

## Использование

Горячая клавиша: `Win + V`
(можно изменить в скрипте).

Примеры использования:
  - Скриншот экрана (клавиша PrtSc)
  - Копировать картинку с браузера
  - Копировать выделенную часть в графическом редакторе
Затем переключитесь в окно проводника или откройте диалог `открытия файла` в любой программе.
Нажмите `Win + V` — PNG-файл будет сохранён в активную папку. Тут же можете его открыть — Удобно!


## Как изменить горячую клавишу

Откройте файл `SaveClipboardImageToActiveFolder.ahk` в любом текстовом редакторе (Блокноте).
Найдите в самом начале строку, которая заканчивается двумя двоеточиями `::` (по умочанию `#v::`).
Замените символы слева от `::` на нужную комбинацию по правилам:

| Символ | Клавиша           | Пример           | Действие                     |
|--------|-------------------|------------------|------------------------------|
| `#`    | Win (Windows)     | `#v`             | Win + V                      |
| `^`    | Ctrl              | `^s`             | Ctrl + S                     |
| `!`    | Alt               | `!f`             | Alt + F                      |
| `+`    | Shift             | `+v`             | Shift + V                    |
| `^!`   | Ctrl + Alt        | `^!c`            | Ctrl + Alt + C               |
| `#+`   | Win + Shift       | `#+s`            | Win + Shift + S              |
| `<!`   | левый Alt         | `<!1`            | Левый Alt + 1                |
| `>!`   | правый Alt        | `>!2`            | Правый Alt + 2               |

Комбинации можно комбинировать, например:
`^#f` → Ctrl + Win + F
`+!d` → Shift + Alt + D

**Важно:**
- Регистр букв не имеет значения (v или V — одно и то же).
- Не используйте пробелы внутри комбинации.
- Если нужная клавиша сама по себе (без модификаторов) — пишите её как есть: `F1` или `Numpad0`.

После изменения сохраните файл и **перезапустите скрипт** (правый клик по иконке в трее → Reload).

## Удаление

### Быстрое удаление через горячую клавишу
- Нажмите `Ctrl + Win + Delete` (удерживая Ctrl и Win, нажмите Delete).
- Подтвердите действие — скрипт удалит себя из автозагрузки и завершит работу.
- Сам файл `.ahk` останется на месте, его можно запустить заново или удалить вручную.

### Удаление вручную
- Нажмите `Win + R` → `shell:startup` → удалите ярлык `SaveClipboardToActiveFolder.lnk`
- Если забыли где лежит основной скрипт, то
(иконка в трее → Edit → В блокноте `Сохранить как...` → Скопировать путь в строке адреса)
- Остановите скрипт (иконка в трее → Exit)
- При необходимости удалите исходный скрипт.
